SpitOclock

A visual LED programmer for your NeoPixel ring clock

What is SpitOclock?

SpitOclock turns a Raspberry Pi Zero and two NeoPixel LED rings into a programmable clock you control from your browser.

The clock has two LED rings:

  • Outer ring — 36 LEDs that display the clock hands (hour, minute, second) with customizable colors
  • Inner ring — 15 LEDs for ambient illumination, effects, or solid colors

You design programs on your computer, preview them live, and push to the clock with one click.

1 Install

Get SpitOclock running on your computer in a few minutes.

What you need

  • Python 3.10+ — python.org/downloads
  • Node.js 18+ — nodejs.org (LTS)
  • The SpitOclock Pi Zero plugged in and on the same WiFi

Windows

1
Install Python — download from python.org. Check "Add Python to PATH" during install.
2
Install Node.js — download LTS from nodejs.org, click through defaults.
3
Copy the spitclock folder to your Documents folder, open a terminal, and run: 
cd %USERPROFILE%\Documents\spitclock
scripts\install.bat
4
Launch! 
spitoclock
Opens http://localhost:8421 in your browser.

Mac / Linux

git clone https://github.com/theomaire/spitclock.git
cd spitclock
./scripts/install.sh
spitoclock

3 Programs

A program defines what the clock looks like. Create as many as you want.

Clock

Shows hour, minute, and second hands on the outer ring with custom colors and brightness.

Solid

All LEDs light up in a single color. Great for ambient lighting.

Effect

Animated patterns: rainbow cycle, breathing, chase, sparkle, or fire.

How to create a program

1
Type a name in the "New program ID..." field and click + New
2
Choose a Type (clock, solid, or effect)
3
Adjust colors and brightness using the color pickers and sliders. The live preview updates instantly.
4
Click Save, then Push to Clock to send it to the Pi
Each program controls both rings independently

Outer Ring (Clock)

  • Brightness
  • Hour / Minute / Second colors
  • Background color
  • Effect type & speed (if effect mode)

Inner Ring (Illumination)

  • Mode: off, solid, rainbow, breathing, chase
  • Brightness (0–255)
  • Color
  • Speed

Setting up a schedule

1
Switch to the Schedule tab
2
Pick a default program (runs 24/7 as the baseline)
3
Add time rules — e.g., "Night Mode" from 22:00 to 07:00. Optionally filter by day of week.
4
Push to Clock — the Pi will automatically switch programs at the scheduled times

Chimes

Like a church bell, chimes briefly flash a program at regular intervals:

  • Every hour (:00), half hour (:30), or quarter hour (:15, :45)
  • Set the duration (how many seconds the chime lasts)
  • Restrict to active hours (e.g., 8:00–22:00, so it doesn't flash at night)

5 Push to Clock

One click sends everything to the Pi.

The Push to Clock button in the top-right corner uploads your config to the Pi and restarts the clock service.

The connection indicator shows:

  • Green — connected, shows latency
  • Red — not connected

The app auto-discovers the Pi via Tailscale or local WiFi (mDNS).

6 Troubleshooting

"Cannot connect to Pi"

  • Make sure the Pi is plugged in and has had 30+ seconds to boot
  • Check your computer and Pi are on the same WiFi
  • Click Refresh in the connection panel

"python is not recognized" (Windows)

Reinstall Python and check "Add Python to PATH" during installation.

"spitoclock is not recognized"

Close and reopen your terminal. If it still fails, run python -m server.app from the spitclock folder.

Updating SpitOclock

Replace the spitclock folder with the new version and re-run the installer script.
